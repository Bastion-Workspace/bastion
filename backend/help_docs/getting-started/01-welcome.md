# Welcome

Bastion is a personal knowledge workspace that combines documents, chat, and AI agents in one place. You store files in the **Document Library**, talk to built-in and custom agents in **Chat**, and design workflows in **Agent Factory**. This page introduces the main areas and how to get around.

---

## Main areas

- **Documents** — Your files and folders live here. The left sidebar shows **My Documents**, **Teams**, and **Global Documents**. You create folders, upload files, and open documents in tabs. Markdown and Org Mode files are first-class; PDFs, images, and other types open in appropriate viewers.
- **Chat** — Open the chat sidebar from the right edge of the screen (or the **Open Chat** button when the sidebar is collapsed). You send messages and get responses from agents; you can @mention custom agents from Agent Factory. Conversations are listed in the sidebar; you can create, rename, pin, and delete them.
- **Agent Factory** — Create custom agents with playbooks, tools, and data connections. Each agent has an @handle so you can invoke it in chat. You don’t need to write code; you compose steps in the Workflow Composer.
- **Teams** — Create or join teams for shared documents and team feeds (posts and comments). Team documents appear in the Document Library under each team.
- **Settings** — Configure your profile, appearance, model preferences, external connections (email, messaging bots), Org-Mode options, media sources, and more. Open **Settings** from the user menu in the top bar.

Other entries in the main navigation (e.g. **News**, **Map**, **Media**, **Games**) may appear depending on your instance and your permissions.

---

## Navigating the interface

- **Top bar** — Main navigation (Documents, Teams, Media, Agent Factory, etc.), user menu (theme, settings, help, logout), and icons for notifications and messaging.
- **Sidebars** — On the Documents page, the left sidebar is the folder tree and org-related views (All TODOs, Contacts, Tags Browser, RSS feeds, **Data Workspaces**). On the right, the chat sidebar shows conversations and the current chat. You can resize or collapse the chat sidebar.
- **Tabs** — Documents and other content open in tabs in the center area. You can have several tabs open and switch between them.
- **Status bar** — At the bottom: date and time (your timezone), weather, music controls (if media is configured), and app version.

---

## Quick Capture

From anywhere in the app you can press **Ctrl+Shift+C** (or **Cmd+Shift+C** on Mac) to open **Quick Capture** — add a quick note, TODO, contact, or meeting to an Org file without opening the file. For **journal entries**, press **Ctrl+Shift+J** (or **Cmd+Shift+J** on Mac) to open **Journal for the day** and view or edit the day’s entry. See **Org Quick Capture** in the Chat and Agents section for details.
