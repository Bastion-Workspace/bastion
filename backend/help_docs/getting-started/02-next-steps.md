# Next steps

After setup, use this checklist to get the most out of Bastion. You can do these in any order; the list is a suggested path from basics to deeper features.

---

## Create and organize content

- **Create a folder** — In the Document Library, right-click **My Documents** (or any folder) and choose **New User Folder** or **New Folder**. Name it and use it to group your files.
- **Upload a file** — Right-click a folder and choose **Upload Files**. Select one or more files from your computer; they appear in that folder. Supported types include Markdown (`.md`), Org Mode (`.org`), PDFs, and images.
- **Create a Markdown or Org file** — Right-click a folder and choose **New Markdown** or **New Org-Mode**. Name the file when prompted. Use Markdown for notes and docs; use Org for outlines, TODOs, agendas, and contacts.

---

## Use chat and agents

- **Open chat** — Click the chat icon or the right-edge toggle to open the chat sidebar. Type a message and send; you get a response from the default agent. You can switch conversations from the list or create a new one.
- **Try Quick Capture** — Press **Ctrl+Shift+C** (or **Cmd+Shift+C** on Mac) from anywhere. Choose a capture type (e.g. Note or TODO), pick a target Org file and optional heading, then save. For journal, use **Ctrl+Shift+J** to open Journal for the day.
- **Mention a custom agent** — If you have Agent Factory agents, type **@** in the chat input and pick one by name. The agent runs its playbook in the context of your message.

---

## Configure settings

- **Models** — In **Settings > Models**, choose which model to use for chat, fast replies, and image generation. Add **User LLM Providers** if you use your own endpoints.
- **Appearance** — In **Settings > Appearance**, set light or dark theme and accent color.
- **Connections** — In **Settings > Connections**, add email (Microsoft OAuth or IMAP/SMTP) and messaging bots (Telegram, Discord, Slack, etc.) so agents can send and receive messages and email.
- **Org-Mode** — In **Settings > Org-Mode**, configure TODO keywords, tags, and agenda display to match your workflow.

---

## Discover more features

- **Teams** — Go to **Teams** in the nav to create a team or accept an invitation. Team documents show up in the Document Library under the team name; team feeds let you post and comment.
- **RSS feeds** — In the Document Library sidebar, open **RSS** and add feeds. Articles open in a tab; you can search them like other content.
- **Data Workspace** — In the sidebar under Data Workspace, create tables and import data. Agents and playbooks can query this data with SQL tools.
- **Knowledge Graph** — Open the **Entity Relationship Graph** or **File Link Graph** from the document tabs to browse entities and file links. Agents can populate and use the graph.
- **Help** — Open **Help** from the user menu to read all in-app documentation. You can pop the help panel out and drag it around.
